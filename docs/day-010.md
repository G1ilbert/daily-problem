# Day 010 — Click Logging + Stats

## โจทย์วันนี้
ไม่รู้ว่า link ไหนถูกคลิกกี่ครั้ง

## แก้ยังไง
เพิ่ม write_log ใน find_url เขียน datetime + code ลง log.txt ทุกครั้งที่มีคนคลิก
เพิ่ม GET /stats/:code อ่าน log.txt แล้วนับ code ที่ตรง
ถ้า code ไม่มีใน mapping → 404

## เรียนรู้อะไร
fscanf ต้องระวัง space ใน datetime
find_url_no_log แยกออกมาเพื่อไม่ให้ stats เพิ่ม log ผิด
