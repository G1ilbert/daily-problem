# Day 011 — Validate URL

## โจทย์วันนี้
รับ URL อะไรก็ได้เข้าระบบ ไม่มีการตรวจสอบ
อยากให้ตรวจสอบก่อนว่าเป็น URL จริงๆ

## แก้ยังไง
สร้าง validator.c แยกออกมาทำหน้าที่ตรวจสอบอย่างเดียว
ใช้ regex ตรวจว่าขึ้นต้นด้วย http:// หรือ https:// และมี domain จริงๆ
handle_post เรียก is_valid_url ก่อน ถ้าไม่ผ่านคืน 400

## เรียนรู้อะไร
regex.h มีใน Linux built-in ไม่ต้องติดตั้งเพิ่ม
แยก validator ออกมาทำให้ test และแก้ได้ง่ายกว่าเขียนรวมใน handler
trailing slash ทำให้ idempotency พังได้
