# Day 001 — Basic HTTP Server + Redirect

## ใช้อะไรทำ
- C
- BSD Socket API

## ทำงานยังไง
browser ส่ง request มา → server อ่าน → ส่ง 302 กลับไป → browser ไปหน้าใหม่

## โจทย์วันนี้
ทำให้ server รับ request แล้ว redirect ไปยัง URL ได้

## แก้ยังไง
ส่ง HTTP response 302 กลับไปพร้อม Location header
browser เห็น 302 แล้วไปเองอัตโนมัติ
