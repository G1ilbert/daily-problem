# Day 005 — perror + Error Messages

## โจทย์วันนี้
เปลี่ยน printf เป็น perror แล้วดูว่า OS บอกอะไรเพิ่ม

## แก้ยังไง
นำ perror มาแทนการ printf เฉยๆ

## เรียนรู้อะไร
perror    → อ่านค่า errno แล้วแปลงเป็นข้อความ
errno     → OS เก็บ error code ไว้ทุกครั้งที่ล้มเหลว
strerror  → แปลง errno เป็น string เอง
