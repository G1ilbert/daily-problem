# Day 004 — Error Handling

## โจทย์วันนี้
mapping.txt หาไม่เจอ server ยังรันต่อเงียบๆ
อยากให้หยุดทันทีและบอกสาเหตุ

## แก้ยังไง
จำลองหา server ไม่เจอแบบง่ายๆจากการเปลี่ยนชื่อ ไปค้นหา mapping1.txt ที่ไม่มีอยู่จริง
เพิ่ม exit(1) ตอนที่ fopen คืนค่า NULL
ต้อง include stdlib.h ด้วยถึงจะใช้ exit ได้

## เรียนรู้อะไร
exit(1) หยุด program ทันที code 1 หมายถึง error
ถ้า server รันต่อโดยไม่มี mapping ทุก request ได้ 404 หมดโดยไม่รู้สาเหตุ
