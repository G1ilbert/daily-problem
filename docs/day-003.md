# Day 003 — Separate Routing Module

## โจทย์วันนี้
mapping hardcode อยู่ใน code
อยากเพิ่ม link ต้อง compile ใหม่ทุกครั้ง
เปลี่ยนให้อ่าน mapping จาก file แทน

## สิ่งที่ทำ
สร้าง mapping.txt เก็บ code กับ url แต่ละบรรทัด
สร้าง route.c เก็บ logic การอ่าน file และหา url
สร้าง route.h ประกาศ function ไว้เป็น interface
server.c include route.h แล้วเรียกใช้ load_routes และ find_url
