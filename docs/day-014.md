# Day 014 — Graceful Shutdown

## โจทย์วันนี้
Ctrl+C หยุด server ทันทีโดยไม่ close socket
รันใหม่เร็วๆ อาจเจอ bind failed: Address already in use

## แก้ยังไง
เพิ่ม SO_REUSEADDR บอก OS ให้ใช้ port เดิมได้ทันที
เพิ่ม signal handler ดัก SIGINT ให้ close server_fd ก่อนหยุด
ย้าย server_fd เป็น global ให้ signal handler เข้าถึงได้

## เรียนรู้อะไร
TIME_WAIT ไม่ใช่ bug คือ feature ของ TCP
ป้องกัน packet เก่าที่ค้างอยู่ใน network
SO_REUSEADDR บอก OS ว่า application handle TIME_WAIT เองได้
signal handler คือวิธีที่ application รับรู้ว่า OS ส่ง signal อะไรมา
